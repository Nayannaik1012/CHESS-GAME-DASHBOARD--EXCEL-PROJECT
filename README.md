# CHESS-GAME-DASHBOARD--EXCEL-PROJECT
Technologies: Microsoft Excel (Pivot Tables, Charts, Slicers, Dashboard Design) 
•	Created an interactive Microsoft Excel dashboard to analyze chess game statistics across multiple matches. 
•	Used pivot tables and slicers to allow dynamic filtering by player color, game result, date, opening patterns, and opponent type. 
•	Visualized key metrics including win rate, most common openings, time-based performance trends, and victory/defeat breakdown. 
•	Enabled deeper analysis by adding KPI indicators, conditional formatting, and automated insights for strategy improvement.
